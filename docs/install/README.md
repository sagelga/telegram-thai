# ใช้ภาษาไทย
เพื่อใช้ Telegram เป็นภาษาไทย เพื่อนๆ จำเป็นที่จะต้องกดเข้าไปยังลิงก์ด้านล่าง โดยลิงก์จะทำการเปิดแอพ Telegram ที่อยู่บนอุปกรณ์ของท่านและแสดงตัวเลือกในการเปลี่ยนภาษาเป็นภาษาไทย
